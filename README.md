# kosh-learn

## Bourne家族 shell    (Bash 和 zsh 在一定程度上支持C Shell)
- Bash
  - Bourne Again    (Linux的标准)
- zsh
- sh
  - Bourne Shell
- ksh
  - Korn Shell
- psh
  - POSIX Shell

## C家族 shell
- csh
  - C Shell
- tcsh
  - TC Shell

## 查看shell的支持和类型

```shell
cat /etc/shells    # 查看当前系统支持的shell
echo $SHELL   # 当前 shell 类型
```
